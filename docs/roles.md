# Роли и права доступа

## Глобальные роли

| Роль | Чтение своих чатов | Отправка сообщений | Администрирование |
|---|---:|---:|---:|
| `reader` | Да | Только в «Избранном» | Нет |
| `writer` | Да | Да | Нет |
| `admin` | Да | Да | Да |

## Роли внутри чата

| Роль | Чтение | Отправка | Управление чатом |
|---|---:|---:|---:|
| `readonly` | Да | Нет | Нет |
| `member` | Да | Да | Нет |
| `owner` | Да | Да | Да |

## Принцип минимальных привилегий

Пользователь видит только те чаты, где он есть в `chat_members`. Поиск сообщений также ограничен доступными чатами.

`reader` может читать обычные чаты, копировать сообщения и добавлять их в «Избранное». В обычных чатах поле отправки заблокировано. В «Избранном» читатель может писать заметки себе, отвечать, удалять сообщения из своего избранного и редактировать только собственные обычные заметки. Пересланные в «Избранное» сообщения редактировать нельзя.

Управление участниками группового чата доступно только глобальному `admin` или владельцу конкретного чата (`owner`). Остальные пользователи видят только список участников и предупреждение о недостатке прав.

«Избранное» является личным чатом пользователя, поэтому добавлять или удалять участников в нем нельзя.

## Обновление: права для файловых сообщений

Файловые сообщения подчиняются тем же правилам доступа, что и обычные текстовые сообщения:

- `admin` и `writer` могут отправлять текстовые и файловые сообщения в доступные им чаты;
- `reader` может читать обычные чаты, добавлять сообщения с файлами в «Избранное», но отправлять сообщения может только в «Избранном»;
- участник группы с ролью `readonly` может просматривать файловые сообщения, но не может отправлять новые сообщения и файлы;
- управление участниками группы не связано с вложениями и остается доступным только администратору/владельцу группы.

При пересылке файлового сообщения в «Избранное» пользователь получает личную копию сообщения с вложениями. Если пользователь не является участником исходного чата, переход из «Избранного» к исходному сообщению блокируется уведомлением.
